# DanaKita - Portal Keuangan Komunitas

Aplikasi pengelolaan kas komunitas (RT/RW/Organisasi) yang mudah digunakan, aman, dan transparan.

## 🚀 Cara Menjalankan Aplikasi

Aplikasi ini dapat dijalankan langsung tanpa perlu instalasi tambahan (Node.js/NPM).

1. **Unduh Kode:** Unduh seluruh file proyek ini.
2. **Buka di Browser:** Klik dua kali pada file `index.html` untuk membuka Portal Warga, atau `super-admin.html` untuk Panel Super Admin.
3. **Login:** Gunakan ID Komunitas (Contoh: `DIR-01`) untuk masuk ke dalam sistem.

## 🛠 Teknologi yang Digunakan
- **HTML5 & CSS3:** Struktur dan desain aplikasi.
- **Tailwind CSS:** Framework CSS untuk UI yang responsif.
- **Supabase:** Backend as a Service untuk Database dan Auth.
- **Phosphor Icons:** Set ikon untuk antarmuka pengguna.
- **html2pdf.js:** Untuk mencetak bukti pembayaran ke PDF.

## 🔒 Keamanan & Privasi
- Data disimpan dengan aman di database Supabase.
- PIN Warga dienkripsi sebelum disimpan.
- Akses Admin dilindungi dengan sistem otentikasi tingkat lanjut.
